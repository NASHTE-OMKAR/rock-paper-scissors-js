# Rock Paper Scissors Game

A browser-based implementation of the classic Rock Paper Scissors game, built with vanilla JavaScript during daily coding practice for Salesforce LWC development.

## Features
- Play against computer AI using random selection
- Real-time score tracking for player and computer
- Visual feedback for choices and game results
- One-click reset to start over
- Responsive circular buttons with hover effects

## Technologies Used
- **HTML5**: Structure and content
- **CSS3**: Styling, layout (Flexbox), animations
- **JavaScript (ES6+)**: 
  - DOM manipulation (`querySelector`, `addEventListener`)
  - Event handling (click, reset)
  - Conditional logic and game rules
  - Random number generation

## How to Run
1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Click on Rock, Paper, or Scissors to play!
4. Use the RESET button to restart

## Learning Goals
This project was built as part of a 30-day JavaScript bootcamp focused on foundational skills for **Salesforce Lightning Web Components (LWC)** development, emphasizing:
- User interaction handling
- State management (scores)
- Dynamic UI updates
- Clean, modular code structure
